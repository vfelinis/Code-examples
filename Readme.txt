File store представляет собой проект для использования на занятиях в обучении студентов.
Функционал приложения сводится к возможности хранения файлов различных форматов. Имеется система регистрации и авторизации.
Пользователи хранятся в MySQL, сохраненные документы размещаются в MongoDB. При входе в личный кабинет у пользователя есть возможность
создать новый документ, изменить или удалить уже имеющиеся. Имеющиеся документы отображаются в виде таблицы с названием документа
в виде ссылки для скачивания, описанием и датой. Есть пагинация и поиск. Имеется клиентская и серверная валидация форм.
У привилегированных пользователей имеется административная панель.
Stage 1: представляет собой использование PHP.
Stage 2: добавляется частичная реализация SPA архитектуры с использованием Knockout.js.
Stage 3: перенос функционала на Node.js, полностью SPA архитектура на основе React, дополнительно используется Redux.
Stage 4: перенос функционала на Asp.Net Core с использованиям AngularJS 1.5
На данный момент реализованы две первые стадии, третья в разработке.
